
# 🧬 MediLoom: Accelerating Medical Knowledge Discovery through Automated Knowledge Graph Generation and Enrichment 

## 📌 Overview
Knowledge Graphs (KGs) are essential for structuring and organizing biomedical knowledge, yet challenges remain in automating their construction and ensuring completeness. Existing approaches often result in fragmented graphs with missing semantic connections.

This project introduces **Medical Knowledge Graph Automation (M-KGA)** — a novel framework that automates KG generation and enrichment using biomedical ontologies and embedding-based inference to uncover hidden relationships.

## 🧩 Key Contributions
- 🔗 **Automated Knowledge Graph Generation** from user-provided medical concepts
- 🌐 **Semantic Enrichment via BioPortal Ontologies**
- 🧠 **Embedding-Based Link Discovery** to identify hidden relationships
- 📊 Two Novel Approaches:
  - **Cluster-Based Method** (group-level inference)
  - **Node-Based Method** (fine-grained connection discovery)
- ⚡ Improves **connectivity and completeness** of biomedical KGs

---

## ⚙️ Methodology
1. Input medical concepts (from EHRs or other sources)
2. Ontology mapping using BioPortal
3. Knowledge graph construction
4. Graph embedding generation
5. Hidden link discovery using:
   - Cluster-based inference
   - Node-based inference
6. Graph enrichment and validation

---

## 📂 Dataset & Evaluation
- Evaluated on **100 frequently occurring medical concepts** from Electronic Health Records (EHRs)
- Demonstrates improved **graph connectivity and semantic richness**

---

## 📈 Key Insights
- Automated KG construction alone is insufficient without enrichment
- Embedding-based approaches effectively uncover **latent biomedical relationships**
- Combining ontology-driven and data-driven methods significantly enhances KG quality
